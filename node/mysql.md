# mySql

## mySql常用命令
```javaScript
    // 断言 
    show databases:查看所有的数据库;

    use databaseName:查看某个数据库;

    select * from tableName:查询某个表;

    select username,nickname from users:查看users表的某几个字段
    
    insert into users(username,`password`,nickname)values('zhangsan','123','张三'):往user表中添加数据(password是关键词)
```
