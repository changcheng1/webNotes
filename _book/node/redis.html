redis

redis基础

安装

    // 启动redis服务
   redis-server;
   // redis命令行操作
   redis-cli;
   // 设置/修改值
   set name 'changcheng'
   // 获取值
   get name
   // 获取所有的key
   keys *
   // node安装redis插件
   npm i redis -D
